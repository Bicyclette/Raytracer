# RAYTRACER

A basic C++ Raytracer accelerated with OpenCL.

## Dependencies

- [VCPKG](https://vcpkg.io/)
- [CMake](https://cmake.org/)

## Build

```
cmake -DCMAKE_TOOLCHAIN_FILE=${PATH_TO_VCPKG}/scripts/buildsystems/vcpkg.cmake -B build -S .
```
